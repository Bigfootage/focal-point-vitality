== Description ==

Focal Point Vitality is a modern, interactive web application built with React and Vite, perfectly integrated into WordPress. This plugin allows you to seamlessly embed your React application into any WordPress site.

= Features =

* Modern React-based application
* Built with Vite for optimal performance
* Fully responsive design for all devices
* Smooth animations and transitions
* Multiple integration methods (shortcode and full-width template)
* Clean, professional design
* Fast loading and optimized performance
* All assets bundled and optimized

= Usage =

**Method 1: Shortcode (Recommended)**
Add `[focal_point_vitality]` to any page or post.

**Method 2: Full-Width Template**
1. Create or edit a page
2. Select "Focal Point Vitality - Full Width" from the Template dropdown
3. Publish the page

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/focal-point-vitality/`, or install through the WordPress plugins screen
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use Settings > Focal Point Vitality to view usage instructions
4. Add the application to a page using the shortcode `[focal_point_vitality]` or by selecting the full-width template

== Frequently Asked Questions ==

= How do I add this to my site? =

You can use the shortcode `[focal_point_vitality]` on any page or post, or select the "Focal Point Vitality - Full Width" template when editing a page.

= Is this plugin responsive? =

Yes, the application is fully responsive and works perfectly on all devices including mobile, tablet, and desktop.

= Will this work with my theme? =

Yes! This plugin is designed to work with any WordPress theme. The shortcode method is compatible with all themes, and the full-width template works with most modern themes that support custom page templates.

= Does this affect my site's performance? =

No! The plugin only loads assets (CSS and JavaScript) on pages where you use the shortcode or template. All other pages remain unaffected.
